Welcome to nMOLDYN on Jupyter!

nMOLDYN is an interactive analysis program for Molecular Dynamics simulations. It is especially designed for the computation and decomposition of neutron scattering spectra, but also computes other quantities. Jupyter notebooks became very popular in the scientific community to analyze data in an interactive way. You will find 4 Jupyter notebooks (see below), each notebook contains a classical analysis. You can download the notebooks from GitHub or even play interactively with the notebooks thanks to My Binder...

Try me on MyBinder

MSD

From a molecular dynamics simulation of 256 water molecules, we compute the Mean Square Displacement (MSD) for the hydrogen and oxygen atoms. See the code on GitHub.

VACF

From a molecular dynamics simulation of 256 water molecules, we compute the Velocity Auto-Correlation Function (VACF) for the hydrogen and oxygen atoms. See the code on GitHub.

DOS

From a molecular dynamics simulation of 256 water molecules, we compute the Density of States (DOS) for the hydrogen and oxygen atoms. See the code on GitHub.

F(q, t)

From a molecular dynamics simulation of 256 water molecules, we compute the coherent intermediate scattering functions. See the code on GitHub.
